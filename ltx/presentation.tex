\item Built-in constructs: memoization, conditionals
\end{itemize}
\item \texttt{.ypl} $\rightarrow$ translation $\rightarrow$ \texttt{.ml} $\rightarrow$ execution

\item Condensed: \texttt{./yapplc program.ypl ; ./program}
